Quick setup for the Astro + Tailwind scaffold

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

3. Open http://localhost:3000 in your browser.

Notes:
- This scaffold includes `astro`, `tailwindcss`, `postcss`, and `autoprefixer` in package.json. Run `npm install` to fetch them.
- If you want to customize Tailwind, edit `tailwind.config.cjs` and `src/styles/global.css`.
